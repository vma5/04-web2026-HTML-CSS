# 🍳 Sabores do Mundo - Portal de Receitas
O Sabores do Mundo é um site estático desenvolvido para amantes da culinária, oferecendo uma interface visualmente rica e organizada para explorar receitas doces e salgadas. O projeto foca-se na experiência do utilizador (UX) e na simplicidade de navegação, utilizando técnicas modernas de design sem a necessidade de scripts complexos.

🚀 Funcionalidades do Site
Página Inicial (Home):

Banner Hero: Uma imagem de boas-vindas em alta qualidade com sobreposição de texto para captar a atenção.

Receita em Destaque: Uma secção especial que apresenta o "Bolo de Cenoura com Chocolate", com uma breve descrição e um botão para abrir a receita completa.

Categorias Organizadas:

Páginas dedicadas para Doces e Salgados, facilitando a localização de pratos específicos.

Visualização de Receitas (Modais Puro CSS):

Ao clicar em "Ver Receita", abre-se uma janela sobreposta (modal) que contém:

Vídeo de Passo a Passo: Integração direta com o YouTube.

Lista de Ingredientes: Organizada e fácil de ler.

Modo de Preparo: Instruções detalhadas etapa por etapa.

Sistema de Login/Registo Visual: Uma página dedicada para simular a entrada de utilizadores na comunidade.

Design Responsivo: O site adapta-se automaticamente a telemóveis, tablets e computadores.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica de todo o conteúdo.

CSS3: Utilizado para todo o design, animações de hover e a lógica dos modais (utilizando a técnica de checkbox e o seletor :checked).

YouTube Embed: Tecnologia para incorporação de vídeos sem pesar no carregamento do servidor.

⚙️ Como Executar o Projeto
Faz o download de todos os ficheiros para uma pasta local.

Garante que as imagens (banner.jpg, image_1.png, image_3.png) estão na mesma pasta que os ficheiros .html.

Abre o ficheiro index.html em qualquer navegador web moderno (Chrome, Firefox, Safari ou Edge).

Navega entre as páginas utilizando o menu no topo do site.
